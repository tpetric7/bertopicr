# Set Up Python Environment for BERTopic

This function sets up a Python environment with all required packages
for using the BERTopic model within the R package. It checks if the
specified environment exists and installs the necessary packages if
needed.

## Usage

``` r
setup_python_environment(envname = "r-bertopic", python_path = NULL)
```

## Arguments

- envname:

  The name of the Python environment. Default is "r-bertopic".

- python_path:

  Optional path to a specific Python executable.

## Value

None
